# Taskly
A simple Task Management tool to manage daily tasks efficiently. Users can add, view, mark done, and delete tasks. Built to practice **HTML, CSS and JavaScript** fundamentals and problem-solving while creating a practical, user-friendly tool.

---

## Features

- **Smart Task Organisation**
  - Add, edit, complete, or delete tasks instantly.
  - Categorise and filter tasks for better focus and workflow management.

- **Progress Tracking**
  - Real-time progress bar updates with completion percentage.
  - Visual statistics for completed vs pending tasks.

- **Offline Persistence**
  - All tasks are stored locally using the **LocalStorage API** ‚Äî works without internet.

- **Animated Modern UI**
  - Smooth transitions, responsive design, and hover effects.
  - Built with Flexbox, Grid, and CSS animations for an intuitive user experience.

- **Interactive Feedback**
  - Real-time pop-ups, confetti animations, and notifications on task completion.

- **Dark Mode Ready**
  - Elegant dark theme with accessible colour contrasts and eye-friendly design.
